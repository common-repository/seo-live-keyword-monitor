Checking keyword density with the SEO Live Keyword Monitor in realtime.


== Description ==

Check the keyword density of your articles with this pretty cool SEO Live Keyword Monitor plugin in realtime. All you have to do is adding tags to your article using the standard WordPress tag box and press save. The SEO Live Keyword Monitor box will now check the current keyword density of the selected tags in realtime while you are writing your article. On-page optimization based on your selected tags has never been easier! 


== Installation ==

1. Upload Folder `seo_live_keyword_monitor` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can also use the WP Plugin-Installer



== Frequently Asked Questions ==

= How to use the Plugin? =

1. Select the tags you want to track for keyword density by adding them to the WordPress tag box
2. Important: Press once 'Save'or 'Refresh' (for published articles) to update your SEO Live Keyword Monitor Box
3. Write your article and track the density in realtime
4. You can also check older articles where tags already have been added
